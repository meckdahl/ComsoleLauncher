COMSOL PROJECT MANAGER - EASY START
====================================

QUICKEST START (Double-click to run):

Windows Users:
  → Double-click: run.bat

Mac/Linux Users:
  → Double-click: run.sh
  (or in terminal: ./run.sh)

Alternative:
  → python launcher.py


WHAT THESE SCRIPTS DO:
- Automatically detect if UV is installed (uses it for 10-100x faster setup)
- Create virtual environment if needed
- Install dependencies (mph, numpy)
- Launch Comsol Project Manager GUI

REQUIREMENTS:
- Python 3.8 or higher
- Comsol Multiphysics installed
- Java Runtime (comes with Comsol)

OPTIONAL (but recommended):
- UV package manager: pip install uv

Once GUI opens:
1. Place .mph files in 'comsol_projects' folder
2. Click Refresh
3. Select a project
4. Click "Quick Run" to execute simulation
   OR
   Click "Advanced Mode" to edit parameters first

For detailed help: See README.md
